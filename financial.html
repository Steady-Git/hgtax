⚠️ 브라우저 환경 CORS 제한 안내

        

          브라우저에서 직접 API 호출 시 CORS 정책으로 인해 차단될 수 있습니다.

          실제 서비스 환경에서는 Node.js / Python 등 백엔드 프록시 서버를 구성하거나,

          공공데이터포털에서 제공하는 서버 측 호출 방식을 사용하세요.

          이 프로그램의 "데모 데이터" 버튼으로 응답 형식과 UI를 미리 확인할 수 있습니다.
        

        
    
          
  1. 공공데이터포털(data.go.kr)에서 해당 API 활용 신청 후 인증키 발급
    2. 
          
  2. 인증키는 반드시 Decoding 키를 사용 (Encoding 키는 URL 인코딩 상태)
    3. 
          
  3. 백엔드 프록시(예: Express.js http-proxy-middleware, FastAPI)를 구성하여 CORS 우회
    4. 
          
  4. 호출 시 serviceKey 파라미터에 Decoding 인증키를 직접 삽입
    5. 
          
  5. 응답의 response.header.resultCode가 "00"이면 정상
    6.
