기업 재무정보 조회

종목 검색 → 재무제표 선택 → 조회하면 Google Sheets에 자동 저장됩니다

⚙️ Google Apps Script 설정 (최초 1회)
Step 1. Google Sheets 새 파일 생성 → 확장 프로그램 → Apps Script
Step 2. 기존 코드 전부 지우고 apps_script.gs 코드 전체 붙여넣기 후 저장
Step 3. 배포 → 새 배포 → 유형: 웹 앱
  • 실행 계정: 나  |  액세스: 모든 사용자(익명 포함)
Step 4. 배포 완료 → 웹 앱 URL 복사 → 아래 입력창에 붙여넣기 → 연결 확인

⚠️ 코드 수정 후엔 반드시 재배포(배포 → 배포 관리 → 수정)해야 적용됩니다.
