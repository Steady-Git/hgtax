🐍 Python Flask · DART Open API

기업 재무정보 조회

파이썬 로컬 서버를 통해 CORS 없이 DART 실데이터를 조회합니다

🐍 서버 실행 안내
1단계 — 라이브러리 설치 (최초 1회만)
pip install flask requests flask-cors

2단계 — 서버 실행
python dart_server.py

3단계 — 브라우저 주소창에 입력
http://localhost:5000

※ HTML 파일을 더블클릭으로 열면 서버 연결이 안 됩니다.
